**Kwork Dark Theme** — это браузерное расширение, которое добавляет комфортную темную тему на сайт Kwork.ru. Создано для тех, кто много времени проводит на бирже фриланса и заботится о своих глазах 👁️

> ⚠️ **Важно**: Расширение находится в стадии активной разработки. Если вы нашли баг или у вас есть предложения — создайте Issue или напишите мне в Telegram.

## ✨ Особенности

- 🎨 **Полноценная темная тема** — все элементы сайта перекрашены в темные тона
- 🌓 **Автоматическая адаптация** — тема применяется ко всем страницам Kwork без исключения
- 🚀 **Легковесность** — расширение весит всего несколько килобайт и не тормозит браузер
- 🔧 **Простота** — никаких сложных настроек, установил и пользуешься
- 🎯 **Аккуратный дизайн** — цвета подобраны так, чтобы не резать глаз и не мешать работе
- 🔒 **Безопасность** — расширение не собирает никакие данные и не требует доступа к вашим аккаунтам

## 📦 Установка

### Способ 1: Установка из магазина расширений (рекомендуется)

> 🚧 Расширение еще не опубликовано в магазинах. Ссылки появятся после проверки модераторами.

### Способ 2: Ручная установка (для разработчиков и тестировщиков)

#### Для Chrome / Edge / Яндекс.Браузер / Opera:

1. **Скачайте расширение**
   ```bash
   git clone https://github.com/your-username/kwork-dark-theme.git
   ```
   или скачайте ZIP-архив и распакуйте его

2. **Откройте страницу расширений**
   - Chrome/Edge: введите в адресной строке `chrome://extensions/`
   - Яндекс.Браузер: введите `browser://extensions/`
   - Opera: введите `opera://extensions/`
     Включите режим разработчика
  
3. **Включите режим разработчика**

   Найдите переключатель "Режим разработчика" (обычно в правом верхнем углу) и включите его

4. **Загрузите расширение**
   - Нажмите кнопку "Загрузить распакованное расширение"
   - Выберите папку `dark-kwork`, которую вы скачали

5. **Готово! 🎉**
    - Иконка расширения появится на панели инструментов
    - Зайдите на Kwork и наслаждайтесь темной темой

#### Для Firefox:
1. Скачайте расширение (см. шаг 1 выше)

2. Откройте Firefox и введите в адресной строке `about:debugging`

3. Нажмите "Этот Firefox" в левом меню

4. Нажмите "Временное расширение"

5. Выберите файл `manifest.json` из папки с расширением

### Технические детали
Расширение использует **content scripts** — скрипты, которые внедряются на страницы Kwork и применяют пользовательские CSS-стили. Основная магия происходит в файле `dark-theme.css`, где переопределяются стандартные цвета сайта.

### Как внести изменения
1. Форкните репозиторий (нажмите Fork в правом верхнем углу GitHub)
2. Склонируйте свой форк: 
    ```bash
    git clone https://github.com/sweetconsole/dark-kwork.git
    cd kwork-dark-theme
    ```
3. Создайте ветку для ваших изменений:
    ```bash
    git checkout -b feature/your-feature-name
    ```
4. Внесите изменения и закоммитьте их:
    ```bash
    git add .
    git commit -m "Add some feature"
    ```
5. Запушьте изменения:
    ```bash
    git push origin feature/your-feature-name
    ```
6. Создайте Pull Request на GitHub
